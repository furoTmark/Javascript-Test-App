# Javascript-Test-App
In order to run it follow these steps:

1. Set proxy  
   `npm config set proxy http://proxy.abc.com:3128`  
   `npm config set https-proxy http://proxy.abc.com:3128`  

2. Install npm packages  
   `npm install`  

3. Start server  
   `node server.js`  

4. View application  
   `localhost:1337`
